# Nova Dashboard — Технический бриф

## Описание проекта

MVP-дашборд для мебельного салона «Рэлан» (г. Лысва, Пермский край). Портфолио-проект джуна-системного аналитика.

**Деплой:** https://nova-light-alya-s-projects6.vercel.app

## Ключевые задачи

### 1. Авторизация
- Два пользователя: Администратор (полный доступ) и Елена Морозова (менеджер)
- Mock-логин без бэкенда (fallback в catch)
- Сессии в localStorage (`nova_light_auth`)
- Автоматическое переключение ролей

### 2. Пайплайн заказов (Канбан)
- 5 статусов: новый → утверждён → в сборке → готов к выдаче → выдан
- Drag-and-drop между колонками
- Горизонтальная прокрутка колёсиком мыши (без всплытия на страницу)
- Фильтры: все / мои / просроченные
- Номера заказов формата Л/XXX/26

### 3. Seed-данные
- 7 заказов, 7 материалов, 19 задач уборки, 8 командных задач
- Касса: смена 10 000₽, 3 продажи, 1 расход
- Все данные генерируются при первом запуске

### 4. ИИ-штурман Борис
- Мок-режим (без API) + интеграция с Gemini
- Контекстные подсказки на всех 13 разделах
- Онбординг-туториал при первом входе
- Демо-ключ: `nova2026`

### 5. База знаний (Wiki)
- 6 вкладок: О компании, Обучение ПО, Шпаргалки, Баги, Предложения, Pro100 Модули
- Редактирование контента администратором
- Шпаргалки по Инфопредприятию (оплата картой/наличными, счёт)

### 6. Адаптивный дизайн
- Десктоп: полный сайдбар
- Планшет: иконки-бар
- Мобайл: бургер-меню
- Тёмная/светлая темы
